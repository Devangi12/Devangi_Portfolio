KrishiRakshak AI
Edge-AI powered farm decision support system for Indian smallholder farmers. Fuses real-time soil moisture, temperature, humidity, and light intensity data with a lightweight ML model on Raspberry Pi 5 for offline crop stress prediction and irrigation risk assessment. Translates sensor data into actionable advisories — not raw numbers — with optional voice alerts for low-literacy users.
Contributions: Embedded sensing pipeline, sensor integration, system architecture, AI pipeline design.

Industry 4.0 Lab · PDEU
Smart IoT-Based Machine Health Monitor
Real-time predictive maintenance system using ESP32, DS18B20, MPU6050, and ACS712. Implements a hybrid detection algorithm combining fixed threshold logic with mean-based adaptive normalisation — classifying machine health as Normal, Warning, or Fault. Transmits data to ThingSpeak cloud for remote dashboards. Zero backlogs, 3-level LED+buzzer alert system, and fault-tolerant cloud uploads.
ESP32 DS18B20 MPU6050 ACS712 ThingSpeak Arduino C++ WiFi IoT

Research Project
Cognitive Overload Wristband
Non-invasive wearable that detects cognitive overload in real time using multimodal physiological sensing. Fuses EDA/GSR (35–40%), HRV via PPG (30–35%), skin temperature (15–20%), and accelerometer (10–15%) into a Cognitive Load Index — classifying mental state from Calm to Overload. Edge inference on ESP32 with LED and vibration feedback, sampling at rates validated by Nature 2024 dataset.
ESP32 MAX30102 Grove GSR MLX90614 MPU6050 Sensor Fusion

Summer Collaboration · Team SensorSurge
LoRa Military Pager System
Secure, infrastructure-independent wireless pager system using ESP32 and LoRa SX1278 at 433 MHz. Two-node relay architecture through a central base station with structured packet protocol, loop prevention, and LCD feedback. Designed for field deployment in military, disaster, and remote industrial contexts where GSM and internet infrastructure is unavailable.
ESP32 LoRa SX1278 433 MHz LCD I2C Arduino IDE Wireless Protocol

AI-Assisted Health & Safety Wearable
Biomedical monitoring wearable targeting rural healthcare and elderly assistance. Integrates MAX30102 for SpO₂/PPG, DS18B20 for body temperature, MPU6050 for interrupt-based fall detection, and embedded AI logic for anemia risk prediction. Offline-capable low-power design with emergency alert system — designed for practical healthcare accessibility, not just demonstration.

Real-Time Depth Estimation (MiDaS)
Monocular depth estimation pipeline running MiDaS neural network on Raspberry Pi for real-time scene understanding from a single camera feed.
